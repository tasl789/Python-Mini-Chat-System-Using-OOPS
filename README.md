<h1>Python Chat Room System</h1>

<p>
This is a simple Python mini project based on Object-Oriented Programming (OOP).
It allows users to join a chat room, send messages, view chat history, and leave the room.
</p>

<h2>Features</h2>
<ul>
  <li>Create users</li>
  <li>Join and leave chat room</li>
  <li>Send messages</li>
  <li>View chat history</li>
</ul>

<h2>Concepts Used</h2>
<ul>
  <li>Classes and Objects</li>
  <li>Class and Instance Variables</li>
  <li>Constructor</li>
  <li>OOP Basics</li>
</ul>

<h2>Technology</h2>
<ul>
  <li>Python</li>
</ul>
